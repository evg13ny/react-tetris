Tetris game made with React.

For start:
* git clone [repository url]
* cd [local repository]
* delete package-lock.json
* npm install
* npm start

Keys:
* Up - Rotate
* Down - SlowDrop
* Left - Left
* Right - Right
* Q - Quit
* P - Pause
* Space - FastDrop

You can change logic in hooks/useDropTime and hooks/useGameStats.
